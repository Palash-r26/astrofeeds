# AstroFeeds

Your personalized astrology feed, delivered daily on WhatsApp—based on your zodiac, Moon sign, & Nakshatra.

## Overview

AstroFeeds is a modern, responsive web application designed to offer personalized astrology content to users. The platform allows users to explore daily forecasts, generate a free Kundli, check matchmaking compatibility, and learn more about their zodiac signs, all delivered in a sleek, visually engaging interface.

## Project Structure

The project has been carefully organized to maintain a clean root directory. All assets (images, icons) and styles are housed within dedicated directories.

- `/assets/images/` - Contains all imagery used across the site.
  - `/icons/` - Contains various UI icons, including zodiac signs and forecast imagery.
- `/css/` - Contains the main stylesheet `style.css`.
- `index.html` - The homepage and main landing screen.
- `free-kundli.html` - Generate a personalized Janam Kundli.
- `matchmaking.html` - Check compatibility and Gun Milan.
- `zodiac-signs.html` - Explore in-depth traits for each zodiac sign.
- `forecast.html` - Discover 2025 horoscope predictions.
- `newsfeed.html` - Astrology news and updates.

## Tech Stack

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom variables, Grid & Flexbox layouts, modern animations, fully responsive design.
- **Bootstrap Icons**: Used for standard iconography (e.g., shopping cart, navigation toggles).

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in any modern web browser to view the application locally. No build step or server is required for the static frontend.

## Features

- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
- **Beautiful UI**: Modern glassmorphism, subtle watermark backgrounds, and curated typography.
- **Semantic Structure**: Clean HTML layout for easy maintenance and scaling.
